# Word Search App (kotlin)

## Run

```bash
./gradlew run
```

## Run production

```
./gradlew run > /dev/null 2>&1
```

## Example request

```
curl http://localhost:3020/files/hobbit.txt/searchWord?word=Bilbo
```
